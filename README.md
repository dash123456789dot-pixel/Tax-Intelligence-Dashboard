# WISING Wealth Intelligence - Tax & Income Dashboard

A premium, audit-ready cross-border tax intelligence system designed for individuals with complex financial lives across India and the United States.

## 🚀 Overview

WISING Wealth Intelligence provides a high-fidelity interface for managing global remuneration, tracking equity compensation (RSUs/ISOs), and monitoring cross-border tax risks. It translates complex tax jargon into actionable, plain-English financial insights.

## 🛠️ Key Modules

### 1. Tax Intelligence Dashboard (`index.html`)
- **Global Household Income**: Real-time tracking of worldwide income composition.
- **Cross-Border Tax Certification**: CPA-level audit reports tracing income through Form 1116 (Passive Category).
- **DTAA Bridge Protocol**: Automated calculation of Section 901/903 FTC claims under the US-India Double Tax Avoidance Agreement.
- **Residency Monitoring**: Live calculation of Substantial Presence Test (US) and NRI status (India) with simulation capabilities.

### 2. Income & Employment Intelligence (`income.html`)
- **Global Payday Strategy**: Comprehensive view of worldwide gross vs. net-to-bank income.
- **India Salary Logic**: Optimized calculations for both New and Old tax regimes, including Section 10(13A) HRA exemptions.
- **US W-2 Remuneration**: Integration of pre-tax deductions (401k/HSA), Social Security taxes, and state-specific tax liabilities.
- **Workflow Inbox**: Intelligent nudges for document verification and RSU residency mismatch alerts.

## 🎨 Design Philosophy
- **Rich Aesthetics**: Built with a sleek, dark-mode aesthetic using Tailwind CSS.
- **Glassmorphism**: Modern UI components with subtle transparencies and gradients.
- **Micro-Animations**: Interactive elements that provide feedback and improve engagement.
- **Audit-Ready**: Professional layouts designed for clear financial documentation.

## 📦 Setup & Usage
This is a client-side dashboard utilizing:
- **HTML5/Semantic Tags**
- **JavaScript (Vanilla)**
- **Tailwind CSS (via CDN)**
- **Google Fonts (Inter, Outfit, JetBrains Mono)**

Simply open `index.html` in any modern web browser to begin. Use the sidebar navigation to toggle between modules.
